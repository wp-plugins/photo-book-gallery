A Simple but Amazing Gallery Book for Images using turn.js

== Description ==
A Simple but Amazing Gallery Book for Images where you can add multiple images and sort them as per your need, Builtin media uploader is added. 

== Installation ==
1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Photo Book Gallery' and install it
3. Go to Dashboard > Photo Book and upload images as pages of book
4. Use this shortcode [photo-book] in any page
5. Now visit that page
